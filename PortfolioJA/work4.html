004 : Neon Unreal: Project bossfight

In this project we are working with an entire team to develop this game with multiple developers all working on this game together, the game is still very early in development but the premise is that we'll be making a boss fight level that you can tackle with many different unique mechanics.

not a lot of thing are set in stone and are subject to change but we have settled on a couple of core mechanics, of those core mechanics i was tasked with creating a first person sword attack the player can use to hit weakpons on the boss.

For the sword attack i have made a blueprint to interact with the weakpoint system, made animations for the sword swing that all match up and added effects to make it all tie together and also implemented a sword model made by one of the developers!
